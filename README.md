# 使用阿里云官方js
> 说明：仅将 官方js 发布到 npm 便于 browserify, webpack 等直接使用

安装 

```
npm intall alicdn-js-nc
```

使用

```
import { noCaptcha } from 'alicdn-js-nc'
```
